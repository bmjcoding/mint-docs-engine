## Core Concepts

Before diving in, familiarize yourself with these core concepts:

### Projects

A **Project** is the top-level organizational unit in Acme. Each project has its own API keys, settings, and data isolation.

### API Keys

Every API request must include an API key for authentication. You can create and manage keys in your project dashboard.

### Environments

Acme supports separate **development** and **production** environments, each with their own data and configuration.

<Tip>
  Use environment-specific API keys to keep your development and production data completely separate.
</Tip>
